Описание

В пространстве BAR0 реализованы блоки управления.

Блок PE_MAIN предназначен для идентификации версии ядра и количества блоков управления.

Блоки PE_EXT_FIFO предназначены для управления каналами DMA.

Пространство BAR1 оставлено для регистров приложения.
