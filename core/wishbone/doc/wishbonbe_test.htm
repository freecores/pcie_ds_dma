Прошивка WISHBONE_TEST

Версия документа: 1.0

Общие сведения

Назначение Проверка обмена
 

Список тетрад

№ Адрес Название  Описание
0 0x0000 - 0x0FFF TEST_CHECK_CFG Блок TEST_CHECK
1 0x1000 - 0x1FFF TEST_CHECK_DATA Область данных блока TEST_CHECK
2 0x2000 - 0x2FFF TEST_GEN_CGF Блок TEST_GEN
3 0x3000 - 0x3FFF TEST_GEN_DATA Область данных блока TEST_GEN

Сигналы прерывания и запроса DMA

Блок TEST_CHECK формирует запрос DMAR0.

Блок TEST_GEN формирует запрос DMAR1.

Запрос прерывания не формируется.

Описание

В адресном пространстве BAR1 расположены два блока управления. Блок TEST_CHECK предназначен для проверки тестовой последовательности. Регистры управления блока расположены в области TEST_CHECK_CFG. Обращение к ним возможно только одиночными 32-х разрядными командами. Байтовые операции не поддерживаются. Обмен данными с блоком производится через область TEST_CHECK_DATA. В этой области поддерживаются блочные операции.

Блок TEST_GEN предназначен для формирования тестовой последовательности. Регистры управления - в области TEST_GEN_CFG. Обмен данными - через область TEST_GEN_DATA.
