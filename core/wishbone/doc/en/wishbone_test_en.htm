List of BAR1 resources

№ Адрес

Name

 Description
0 0x0000 - 0x0FFF TEST_CHECK_CFG Block TEST_CHECK
1 0x1000 - 0x1FFF TEST_CHECK_DATA DATA area of the block TEST_CHECK
2 0x2000 - 0x2FFF TEST_GEN_CGF Block TEST_GEN
3 0x3000 - 0x3FFF TEST_GEN_DATA DATA area of the block TEST_GEN

Interrupts and DMA request

Block TEST_CHECK generate a query DMAR0.

Block TEST_GENERATE generate a query DMAR1.

Interrupt request is not  generated.

Description

There is a two control block in the BAR1 area. TEST_CHECK_CFG and TEST_CHECK_DATA checked test sequence. TEST_GEN_CFG and TEST_GEN DATA generation test sequence.

Access to the TEST_CHECK_CFG  and the TEST_GEN_CFG  can only be a single 32th command. Byte operations are not supported.

Access the the TEST_CHECK_DATA and the TEST_GEN_DATA can only burst command via DMA.
